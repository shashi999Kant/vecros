# Full Stack Quiz Application with Admin Feature

## Objective

This project is a Full Stack Quiz Application that allows users to participate in quizzes and provides an admin panel for viewing leaderboard scores and user results. The frontend is built using **React.js**, and the backend can be developed using any framework of your choice, such as **Node.js** or **Django**. The application is designed to showcase clean, scalable, and well-documented code while fulfilling the outlined requirements.

## Features

### User Functionality
- **Quiz Selection**: Users can view a list of available quizzes and select one to participate in.
- **Question Management**:
  - Users will be presented with a series of N questions for each quiz.
  - Each question has four answer options, with only one correct answer.
  - Users can select only one answer per question.
- **User Interaction**:
  - The app provides immediate feedback, showing whether the selected answer was correct or incorrect.
  - After submitting an answer, users can proceed to the next question.
  - Upon quiz completion, the app displays a summary of the user’s final score.

### Admin Functionality
- **Admin Authentication**: A simple login mechanism is implemented to differentiate between normal users and admin users.

## Tech Stack

### Frontend
- **React.js**: The user interface is built using React, providing an interactive experience for quiz-takers and admins.
